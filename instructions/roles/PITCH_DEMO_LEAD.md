# Role: Pitch and Demo Lead

## Mission

Own the hackathon story, judging narrative, demo script, and final presentation.

## Responsibilities

- Prepare pitch deck/script
- Shape the "best use of Agnes API" story
- Coordinate demo flow
- Prepare backup demo
- Prepare Q&A answers
- Keep messaging clear and memorable

## Core Pitch

> FamLens helps visually impaired elderly users stay emotionally included in family photo conversations by turning images into spoken, personal, and conversational descriptions.

## Agnes API Angle

> We use Agnes AI not as a chatbot, but as a multimodal accessibility agent. It sees, reads, explains, answers follow-up questions, and helps the user reply.

## Demo Structure

1. Show problem: family sends image, user cannot fully understand it.
2. Upload photo into FamLens.
3. Show short spoken summary.
4. Ask follow-up question.
5. Read image text.
6. Generate warm reply.
7. Show uncertainty on blurry image.
8. Close with emotional inclusion.

## Key Lines

Opening:

> Family photos are emotional messages. But for visually impaired elderly users, those moments can arrive silently.

Middle:

> Existing tools can describe objects. FamLens focuses on helping users understand the family moment.

Closing:

> FamLens is not just helping users see a photo. It is helping them take part in the conversation.

## Q&A Prep

Likely questions:

- How is this different from Apple or Samsung accessibility tools?
- How do you protect privacy?
- What if the AI identifies someone wrongly?
- Why use Agnes AI API?
- How would this integrate with WhatsApp?
- What is your MVP?
- How would elderly users set this up?

- [ ] 2-minute pitch ready
- [ ] Demo script ready
- [ ] Backup demo ready
- [ ] Q&A answers ready
- [ ] Agnes API value clearly explained
